# Overview

*LogINvisible* is an academic project developed during my secod semester of Software Engineering.
It represent my first large project written in C, focused on Understanding program structure, data modeling with strutcs, and introductory authentication concepts.

Altho the implementation is intentionally basic, the project marks my first exposure to security-related ideas, such as password handling, access control, and activity logging.

## How to Use
When the program starts, a main menu will be displayed with the following options:

a) Register user  
b) View users  
c) Log in  
s) Exit  

The user must select the desired option by typing the corresponding letter and pressing Enter.

The program includes two different menus: one for regular users and another for administrators.  
You are encouraged to explore the program to discover its full functionality.
